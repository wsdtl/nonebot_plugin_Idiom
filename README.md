## 安装方式
- #### 使用pip
```
pip install nonebot-plugin-idiom
打开 nonebot2 项目的 bot.py 文件, 在其中写入
nonebot.load_plugin('nonebot-plugin-idiom')
```
- ### 指令:
```
指令： < 成语接龙 | 成语接龙 + 一个成语 >
```
